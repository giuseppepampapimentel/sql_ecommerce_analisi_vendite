# sql_ecommerce_analisi_vendite
Progetto in SQL completo: database relazionale, query analitiche e report PDF con Report Builder.

## Contenuto del progetto fatto
- Database relazionale `ecommerce_db` (tabelle: cliente, ordine, dettaglio_ordine)
- Query SQL per analisi clienti (numero ordini, totale speso, media per ordine)
- Report in formato `.rdl` (Report Builder)
- Report esportato in PDF
- Relazione del progetto in PDF

## Tecnologie utilizzate per la creazione del progetto
- SQL Server
- T-SQL
- Microsoft Report Builder
